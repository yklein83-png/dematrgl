# Templates DOCX

Ce dossier contient les templates DOCX pour la génération de documents :

- DER - Document d'Entrée en Relation
- KYC - Questionnaire Connaissance Client
- Lettre de Mission CIF
- Déclaration d'Adéquation CIF
- Convention RTO
- Rapport Conseil IAS

Les templates utilisent la syntaxe Jinja2 et sont traités par `python-docx-template`.
